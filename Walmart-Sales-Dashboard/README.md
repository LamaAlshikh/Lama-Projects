# Walmart Sales Power BI Dashboard

## Project Description

This interactive **Power BI dashboard** analyzes Walmart weekly sales and presents store performance, monthly sales trends, holiday impact, and fuel-price changes in a clear and visual format.

## Dashboard Highlights

- Total sales and average weekly sales
- Top-performing store
- Weekly sales trend
- Top 10 stores by sales
- Average weekly sales by month
- Holiday vs. non-holiday sales comparison
- Holiday sales uplift
- Average fuel-price trend
- Filters by year, month, store, and holiday status

## Purpose

The dashboard helps users explore sales patterns, compare store performance, understand seasonal and holiday effects, and identify factors that may influence weekly sales.

## Tool

- Microsoft Power BI

## How to Open

Download `Walmart_Sales_Dashboard.pbix` and open it using Microsoft Power BI Desktop.
